# GEOFlow Wiki Source

这套文件是 GEOFlow GitHub Wiki 的仓库内源稿，按中文和英文镜像结构维护。

目录结构：

- `zh/`：中文 Wiki 页面
- `en/`：English Wiki pages

推荐先发布的页面：

- [中文首页](zh/Home.md)
- [English Home](en/Home.md)
- [什么是 GEOFlow](zh/What-Is-GEOFlow.md)
- [What Is GEOFlow](en/What-Is-GEOFlow.md)
- [GEOFlow 方法论](zh/GEOFlow-Methodology.md)
- [GEOFlow Methodology](en/GEOFlow-Methodology.md)
- [使用边界与内容底线](zh/Principles-and-Content-Boundaries.md)
- [Principles and Content Boundaries](en/Principles-and-Content-Boundaries.md)

使用建议：

1. 仓库内继续维护 `docs/wiki/` 作为源稿。
2. GitHub Wiki 初始化后，按页面名把对应内容同步过去。
3. 中文和英文页面尽量保持结构镜像，避免后续维护分叉。

这套 Wiki 的定位不是参数手册，而是：

- 系统定位说明
- 方法论说明
- 使用边界与内容底线
- 适用场景与部署模式
- 采用路径、生态与路线图
