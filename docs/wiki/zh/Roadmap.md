# 路线图

GEOFlow 接下来的演进方向，主要集中在 3 个层次。

## 近期：1-3 个月

- 持续补齐更多 GEO 相关 Skill
- 增强 CLI 覆盖范围
- 完善模板预览与主题启用能力
- 继续完善后台多语言体验
- 补齐更多公开文档和 Wiki 内容

## 中期：3-6 个月

- AI 搜索数据抓取与分析能力
- 对标页面智能识别、样式分析与复刻
- 更强的内容生成链路与质量控制
- 模板系统的进一步工程化
- 更清晰的 API 和自动化接入能力

## 长期方向

- 多端同步能力
- 更强的内容治理与知识治理能力
- 更完整的站点模板市场 / 模板包体系
- 更成熟的 GEO 内容协同平台能力

有一点需要明确：

路线图并不意味着“功能越多越好”。  
GEOFlow 的长期价值，仍然建立在：

- 真实内容
- 知识资产
- 可维护流程
- 稳定系统能力

如果这些基础不牢，路线图就会失去意义。
