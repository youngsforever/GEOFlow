# GEOFlow Wiki 首页

欢迎来到 GEOFlow Wiki。

GEOFlow 是一套面向 GEO 内容工程、AI 搜索适配与自动化内容运营的开源系统。它不是简单的 CMS，也不是单纯的 AI 写作工具。它更像一套内容操作系统：把模型、素材、任务、审核、发布、前台展示，以及 Skill / CLI / API 协同能力串成一条完整链路。

这个 Wiki 主要回答 6 类问题：

- GEOFlow 到底是什么
- 它适合什么团队和场景
- 它的底层方法论是什么
- 它的内容边界和使用底线是什么
- 应该如何部署和逐步采用
- 未来会往哪里继续演进

如果只用一句话描述 GEOFlow：

> 它是一套把“知识资产建设、AI 内容生成、审核发布、前台呈现和自动化协同”工程化的内容系统。

当前已经可用的核心能力包括：

- 多模型接入与配置
- 素材库统一管理
- 任务调度、队列、Worker 执行
- 草稿、审核、发布工作流
- 面向搜索和 AI 引用的前台输出
- 模板预览、主题切换与主题包机制
- Skill / CLI / API 协同能力

这套系统的目标不是扩大信息噪音，而是帮助团队更高效地维护真实、优质、可持续更新的知识内容。
