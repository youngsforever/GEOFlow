# 核心能力总览

GEOFlow 当前已经形成了比较完整的能力闭环。

## 1. 模型管理

- 多模型接入
- OpenAI 风格接口兼容
- 模型切换与失败重试
- 智能模型切换与优先级机制

## 2. 素材管理

- 标题库
- 关键词库
- 图片库
- 知识库
- 提示词与特殊提示词

## 3. 任务调度

- 创建任务
- 定时调度
- 队列执行
- Worker 运行
- 失败重试
- 执行状态追踪

## 4. 内容流程

- 草稿生成
- 人工审核
- 自动通过
- 发布状态管理
- 回收站与恢复

## 5. 前台输出

- 首页、分类页、归档页、文章页
- SEO 元信息
- Open Graph
- 结构化数据
- 主题预览与模板切换

## 6. 协同与扩展

- Web 后台
- API
- CLI
- Skill 生态
- 模板包机制

这些能力的意义不在于“模块多”，而在于它们已经形成了真实的系统链路：

> 素材 -> 任务 -> 队列 -> 生成 -> 审核 -> 发布 -> 前台 -> 继续维护
