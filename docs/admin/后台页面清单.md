## 测试 / 验证页面

这些页面仅用于开发排查，不属于正式后台功能。源码已移动到 `admin/legacy/`，原路径只保留兼容跳转壳。

- `minimal-test.php`

- `test-dashboard.php`

## 维护约定

- 新功能优先接入正式入口，不要继续往 `simple / backup / test / new / safe` 页面加逻辑。
- 如需保留兼容入口，优先改为显式跳转到正式页。
- 需要做路径改动时，统一通过 `admin_url()` / `admin_redirect()`，不要再写死 `/admin/` 或 `/geo_admin/`。
