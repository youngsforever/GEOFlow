## 🔧 服务器管理

### 查看服务器状态
服务器当前正在运行，你可以在终端看到访问日志。

### 停止服务器
如需停止服务器，在终端按 `Ctrl + C`

### 重启服务器
```bash
# 方法1: 使用启动脚本
./start-server.sh

# 方法2: 手动启动
php -S localhost:8080 router.php
```

### 查看日志
```bash
# 查看应用日志
tail -f logs/$(date +%Y-%m-%d).log

# 查看批量执行日志
tail -f logs/batch_*.log

# 查看任务管理器日志
tail -f logs/task_manager_$(date +%Y-%m-%d).log
```

## ⚠️ 注意事项

### 开发环境提醒
1. 当前使用PHP内置服务器，仅适合开发测试
2. 生产环境建议使用Nginx/Apache
3. 数据库文件请定期备份

### 安全提醒
1. 首次登录后建议修改管理员密码
2. 不要在公网暴露开发服务器
3. 定期检查系统日志

### 性能提醒
1. SQLite适合中小型项目
2. 大量数据建议迁移到MySQL/PostgreSQL
3. 图片建议使用CDN加速

## 🆘 常见问题

### Q: 页面显示空白？
A: 检查PHP错误日志，可能是数据库连接问题

### Q: 无法登录后台？
A: 确认用户名密码正确，检查session配置

### Q: 文章不显示？
A: 检查文章状态是否为'published'

### Q: 批量执行无法启动？
A: 查看系统诊断页面，清理孤儿进程
