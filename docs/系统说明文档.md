## 🔄 主要工作流程

### 内容生成工作流
```
1. 管理员创建任务
   ├─ 选择标题库
   ├─ 选择AI模型和提示词
   ├─ 选择图片库(可选)
   ├─ 设置发布间隔
   └─ 设置草稿限制

2. 任务调度器触发批量执行
   └─ bin/cron.php

3. AI引擎生成文章内容
   └─ AIEngine::executeTask()

4. 文章保存为草稿
   └─ 存入articles表，status='draft'

5. 管理员审核(如需要)
   └─ articles-review.php

6. 自动发布或手动批准
   └─ status='published'

7. 文章显示在前台
   └─ index.php, article.php
```

### 任务执行流程
```
start_task_batch.php (启动批量执行)
    ↓
bin/batch_execute_task.php (后台工作进程)
    ↓
AIEngine::executeTask() (生成单篇文章)
    ↓
TaskStatusManager (管理进程生命周期)
    ↓
日志写入 /logs/batch_*.log
```
