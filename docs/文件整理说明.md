## 🛠️ 整理脚本

整理使用的脚本：`organize_files.sh`

**使用方法**：
```bash
chmod +x organize_files.sh
./organize_files.sh
```

**脚本功能**：
1. 创建新的文件夹结构（docs/, _backups/, _archived/）
2. 移动所有 .md 文档到 docs/
3. 移动所有 .bak 备份文件到 _backups/geo_admin/
4. 整理旧的归档文件夹
5. 创建 README.md 和 docs/README.md
