# GEO+AI 系统 - 快速参考卡片

## 🚀 一键启动

```bash
cd "AI Coding/Tool/GEO网站系统"
./start.sh
```

| 功能 | 地址 |
|------|------|
| 🏠 前台首页 | http://localhost:8080 |
| ⚙️ 管理后台 | http://localhost:8080/geo_admin/ |
| 📊 系统诊断 | http://localhost:8080/geo_admin/system_diagnostics.php |
| 📝 文章管理 | http://localhost:8080/geo_admin/articles-new.php |
| ⚡ 任务管理 | http://localhost:8080/geo_admin/tasks-new.php |
| 🤖 AI配置 | http://localhost:8080/geo_admin/ai-configurator.php |

## 💻 常用命令

```bash
# 环境检查
./check-env.sh

# 启动服务器
./start.sh

# 快速打开浏览器
./open.sh

# 查看日志
tail -f logs/$(date +%Y-%m-%d).log

# 安全检查数据库
php bin/db_maintenance.php check

# 安全备份数据库
php bin/db_maintenance.php backup
```

```bash

ps aux | grep php

du -sh data/backups/

# 查看日志大小

```

## 🐛 常见问题

### 端口被占用
```bash
# 使用其他端口
php -S localhost:3000 router.php
```

### 权限问题
```bash
chmod -R 755 data logs uploads
```

### 数据库问题
```bash
# 检查数据库连接与表数量
php bin/db_maintenance.php check
```
