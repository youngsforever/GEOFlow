```

```

## 🔍 问题3：标题库显示0个标题

### 问题详情

**页面**: https://localhost:8080/geo_admin/task-edit.php?id=23  
**显示**: 少儿编程 (0个标题)  
**实际**: 少儿编程标题库有40个标题

### 根本原因

**问题**: `title_libraries`表的`title_count`字段未同步更新

**数据库状态**:
```sql
-- 修复前
SELECT id, name, title_count FROM title_libraries WHERE id = 2;
2|少儿编程|0

-- 实际标题数
SELECT COUNT(*) FROM titles WHERE library_id = 2;
40

-- 修复后
2|少儿编程|40
```

### 为什么会出现不同步？

**原因**: `title_count`字段需要手动更新，但在以下情况下可能未更新：
1. 直接在数据库中插入标题
2. 批量导入标题时更新失败
3. AI生成标题时更新失败
4. 数据库迁移或恢复时未重新计算

### 已修复

✅ 已执行SQL更新：
```sql
UPDATE title_libraries 
SET title_count = (SELECT COUNT(*) FROM titles WHERE library_id = title_libraries.id) 
WHERE id = 2;
```
